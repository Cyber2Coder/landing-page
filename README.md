# Landing Page

A fully responsive landing page built as part of The Odin Project Foundations curriculum.  
This project demonstrates layout recreation using Flexbox, semantic HTML structure, and clean CSS styling.

## Live Demo
https://cyber2coder.github.io/landing-page/

## Project Structure
- `index.html` — Main page layout  
- `style.css` — Styling and Flexbox layout  
- `script.js` — Placeholder for future interactivity  
- `images/` — Placeholder for future assets

## Features
- Responsive layout using Flexbox  
- Clean, semantic HTML  
- Organized CSS structure  
- Simple, modern landing page design

## What I Learned
- Structuring a page with semantic HTML  
- Using Flexbox for layout and alignment  
- Building a responsive hero section  
- Managing a multi‑file project structure
