# Landing Page

A fully responsive landing page built as part of The Odin Project Foundations curriculum.  
This project demonstrates layout recreation using Flexbox, semantic HTML structure, and clean CSS styling.

## 🔗 Live
https://cyber2coder.github.io/landing-page/

## 📁 Project Structure
- `index.html` — Main page layout  
- `style.css` — Styling and Flexbox layout  
- `images/` — Placeholder for future images  

## 🎯 Features
- Flexbox-based layout  
- Hero section with call-to-action  
- Information card grid  
- Testimonial/quote section  
- Call-to-action banner  
- Footer  
- Responsive behavior on smaller screens  

## 🧠 What I Learned
- Building multi-section layouts with Flexbox  
- Structuring HTML for readability and maintainability  
- Creating consistent spacing and visual hierarchy  
- Using Git and GitHub for version control  
- Deploying static sites with GitHub Pages
